# Curriculum tests (optional)

Place **invariant tests** here that should pass **regardless of which track** (A–D) the learner chose — for example:

- Monorepo graph is valid (`turbo run build` from root).
- `verify-curriculum-structure.sh` exits 0.
- Shared `packages/*` compile.

**Per-track** UI or API tests should live **next to the app** under `apps/<name>/` once those packages exist.

Until the monorepo is generated by `/run-plan`, this folder may stay empty except this README.
