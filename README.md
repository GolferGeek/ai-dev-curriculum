# AI Development Curriculum

Nice work! You've built a monorepo with Phase-00 demo apps and Phase-01 SaaS killers. Now you'll learn to keep them healthy.

## How this works

| Phase | What you learn | Tag |
|-------|---------------|-----|
| ~~00 — Your first AI build~~ | ~~The pipeline: intention → PRD → plan → build~~ | ~~`phase-00`~~ ✓ |
| ~~01 — SaaS killers~~ | ~~Build real apps with auth, databases, and tests~~ | ~~`phase-01`~~ ✓ |
| **02 — Quality engineering** | Scan, fix, monitor, harden, and ship with confidence | `phase-02` ← **you are here** |
| 03 — Research | Understand any codebase and create your own tools | `phase-03` |

Your apps are built. Now you'll learn what professional teams do to **keep them working** — automated error scanning, architecture monitoring, quality gates, and a PR workflow that gets smarter over time.
