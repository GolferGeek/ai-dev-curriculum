# AI Development Curriculum

Welcome back! You've completed Phase 00 — you built a monorepo with working apps using AI agents. Now it's time to build something bigger.

---

## How this works

| Phase | What you learn | Tag |
|-------|---------------|-----|
| ~~00 — Your first AI build~~ | ~~The pipeline: intention → PRD → plan → build~~ | ~~`phase-00`~~ ✓ |
| **01 — SaaS killers** | Build real apps with auth, databases, and tests | `phase-01` ← **you are here** |
| 02 — Quality engineering | Scan, fix, monitor, harden, and ship with confidence | `phase-02` |
| 03 — Research | Understand any codebase and create your own tools | `phase-03` |

Your Phase-00 apps are in `apps/`. Now you'll add **SaaS killer apps** — clones of real products like QuickBooks, Trello, Twitter, and Facebook — with authentication, a real database (SurrealDB), and end-to-end tests.

## Get started

**Read this first:** [docs/phase-01/README.md](docs/phase-01/README.md)

It introduces the new agents, skills, and commands for building SaaS apps.

## When you're done with Phase 01

Move to the next phase:

```bash
git checkout phase-02
```

Your SaaS apps will be there, plus quality engineering tools.
