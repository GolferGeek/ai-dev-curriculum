# AI Development Curriculum

Welcome! This is a hands-on curriculum for learning **AI-assisted development** — building real software by directing AI agents with the right instructions.

You're going to learn by doing: writing intentions, generating plans, and watching AI agents build working apps. By the end, you'll have a monorepo full of apps you built by having a conversation.

---

## How this works

This curriculum is organized in **phases**. Each phase builds on the last:

| Phase | What you learn | Tag |
|-------|---------------|-----|
| **00 — Your first AI build** | The pipeline: intention → PRD → plan → build | `phase-00` ← **you are here** |
| 01 — SaaS killers | Build real apps with auth, databases, and tests | `phase-01` |
| 02 — Quality engineering | Scan, fix, monitor, harden, and ship with confidence | `phase-02` |
| 03 — Research | Understand any codebase and create your own tools | `phase-03` |

Right now you're at the very beginning. There's no app code yet — just documentation and the AI toolkit in `.claude/`. That's intentional. **You're going to build everything.**

## Get started

**Read this first:** [docs/phase-00/README.md](docs/phase-00/README.md)

It walks you through the four commands you'll use and explains everything step by step. No prior AI experience needed.

## When you're done with Phase 00

Move to the next phase:

```bash
git checkout phase-01
```

Your Phase-00 apps will already be there, plus new tools and docs for Phase 01.
