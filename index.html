AI Dev Curriculum · free & open on GitHub

Learn to build real software
by directing AI.

You won't memorize a programming language. You'll learn the skill that's replacing that: describing what you want clearly enough that AI agents build it — then reading, testing, and shipping what they make. That skill survives every framework, every tool, every model upgrade. It's the durable one.

You'll do it, not watch it. Across seven hands-on phases you build a real portfolio — and the rhythm never changes:

say what you intend→spec it→plan it→let the agents build→verify it yourself→ship
Start now — it's free → Pre-class setup (20 min)

Bring a laptop you can install software on. Leave with shipped software and the one skill that won't expire.

Seven phases. One loop. A real portfolio.

Every phase ships step-by-step run orders, the AI agents and skills that do the building, a "demo-grade" quality bar — and plain-language talking points: the concepts you'll be able to explain when you're done.

Two ways to take it

Self-paced (free, start today)

  • Do the 20-minute setup, then:
  • git clone https://github.com/GolferGeek/ai-dev-curriculum.git
  • git checkout -b my-phase-00 phase-00
  • Follow each phase's README and run order; every phase has talking points — and anything you can't explain, your AI agent will, in three sentences.
